Tensorflow MNIST tutorial

Based on Tensorflow MNIST tutorial (with reduced number of features/neurons to limit the size of trained model) : Softmax for beginners and Deep Learning for more advanced users.
Modified source files: softmax and deep learning with extra code to dump trained model to disk.
Training results data: softmax and deep learning.

Model reimplemented from scratch: Softmax in Excel and Deep in Web browser.

Made with: Scite, Visual Studio Code, Python 3.6 with Tensorflow, NodeJs, Excel/VBA, WinMerge, SQLite, Git and Github.
